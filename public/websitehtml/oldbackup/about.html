About CraftChat

CraftChat is a bot designed to log conversations within Minecraft. It was built using Node.js and has successfully logged messages.

The project was originally forked from urFate/Afk-Bot.

The first recorded conversation dates back to March 22, 2024.

The website for CraftChat was developed by Cheezball69420, utilizing the groundwork laid by the original creator, urFate.
